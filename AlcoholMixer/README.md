# 술과 토닉을 얼마나 섞으면 몇 도의 술이 되지?

- 야구장에 5% 이하의 술을 가져가기 위한 간단한 프로그램

# 실행

```bash
python3 main.py
```

## 받아야 하는 입력

- 술의 도수
- 술의 양
- 토닉의 양

## 내놓을 출력

- 넣어야할 술의 양
- 넣어야할 토닉의 양

# 활용할 단순한 수식

\[
C_{\text{final}} = \frac{V_1 \times C_1 + V_2 \times C_2}{V_1 + V_2}
\]
, 
- \(C_{\text{final}}\)은 최종 혼합물의 도수입니다 (예: 5도)
- \(V_1\)은 원래 술의 양
- \(C_1\)은 원래 술의 도수 (예: 40도)
- \(V_2\)는 토닉 워터의 양
- \(C_2\)는 토닉 워터의 도수 (알코올이 없는 경우 0도)
